### 📅 [데일리 브리핑] 핵심 작업: 자동화 데이터 인프라 구축 및 검증
**🚨 중요성 (Why):**
*   **근거 기반의 콘텐츠 전환:** 모든 콘텐츠 기획과 트렌드 분석의 주체가 '직감'이 아닌 '데이터'가 되도록 시스템을 구축해야 합니다.
*   **반복 작업의 자동화:** 썸네일 디자인, 키워드 리서치, 트렌드 분석 등 노동 집약적인 초기 단계를 코드로 대체하여 시간을 확보하는 것이 최우선 목표입니다.
*   **다음 단계 진입 조건:** 이 인프라가 성공적으로 구축되어야만, 레오 에이전트가 실제 데이터를 가지고 후킹 분석을 진행할 수 있습니다.

**🛠️ 작업 주체 및 실행 지침:**

#### 1. 💻 코다리 에이전트 (담당: 인프라 구축 및 API 연결)
*   **목표:** 유튜브 데이터 API v3를 활용한 인증 및 데이터 수집의 초기 스니펫을 완성합니다.
*   **구체적 지침:**
    1.  **API 설정:** YouTube Data API v3의 사용을 전제로, 서비스 계정 인증을 위한 최소한의 초기 Python 스크립트(예: `youtube_auth_test.py`)를 작성합니다.
    2.  **테스트:** 이 스크립트를 사용하여, 특정 채널(유니님의 채널)의 최근 업로드된 영상 목록 5개와 해당 영상의 '조회수/좋아요 수 변화 추이'를 텍스트로 출력하는 테스트를 진행해 주세요.
    3.  **산출물:** 인증 및 테스트 성공을 보여주는 코드 블록과 실행 결과를 공유합니다.

#### 2. 📺 레오 에이전트 (담당: 데이터 기반 전략 검증)
*   **목표:** 코다리가 구축한 API 연결을 가정하고, 실제 데이터 분석의 '프롬프트 가이드라인'을 미리 설계합니다.
*   **구체적 지침:**
    1.  **시뮬레이션:** 코다리가 데이터를 성공적으로 가져올 수 있다고 가정하고, '경쟁 채널 A'의 성공적인 영상 5개에 대한 '제목-키워드-후킹 포인트' 매트릭스를 추출하는 가상의 분석 결과를 받아 처리하는 프롬프트 구조를 설계해 주세요.
    2.  **출력물:** 이 매트릭스를 기반으로, 30~40대 직장인에게 가장 강력하게 어필할 수 있는 '킬링 카피 3가지'를 도출하는 프로세스를 완성합니다.
