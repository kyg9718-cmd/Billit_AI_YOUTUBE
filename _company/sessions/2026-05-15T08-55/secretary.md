### 💡 2. API 키 발급 절차 가이드

이 키들은 개별적으로 발급받아야 합니다. 아래 절차를 따라 진행하시는 것이 가장 빠릅니다.

1.  **Google Cloud Console 접속:** 모든 API 키는 [Google Cloud Platform Console](https://console.cloud.google.com/)에서 관리해야 합니다.
2.  **새 프로젝트 생성:** `유니 크리에이터 프로젝트` 등의 이름으로 새 프로젝트를 만드세요.
3.  **API 활성화:** 필요한 API(YouTube Data API v3, Google Trends API 등)를 프로젝트에 개별적으로 활성화합니다.
4.  **키 생성 및 제한:** '사용자 인증 정보' 섹션에서 API 키를 생성하고, **반드시 서비스 사용처(API)와 IP 주소로 사용 범위를 제한**하여 보안을 강화해야 합니다.
